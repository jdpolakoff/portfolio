Fishwrap is a frontend news briefing application that pulls headlines from hundreds of news sites. Swipe through stories with a "Tinder"-like carousel.

NOTE: This project is in beta mode (bugs remain) and only currently accessible through http protocal. After a production-level release of the next version of the API, the site will be fully launched and available through https protocal.

TECHNOLOGIES: React, jQuery, News API, Git.

Developed in tandem with two other web development immersive students, using Git for version control.

Hangman 2.0 is a modern, jQuery-enabled take on the classic pen and paper game "Hangman." Game is 2-player, with one player entering a word and the other guessing by selecting letters from the letter bank.
